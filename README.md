# Declutter AI 

Declutter AI is a frontend project designed to help users organize and simplify their digital space. The goal is to eventually integrate AI to make this process smarter and more automated.

## Features (Frontend only as of now) 
- Simple, clean UI to visualize and manage digital clutter
- Sections for organizing files, notes, and more
- Responsive design for better accessibility

##  Tech Stack
- HTML
- CSS
- JavaScript

##  Future Scope
- Integrate AI suggestions (e.g., auto-sort, duplicate detection)
- Add backend using Flask 
- Save user preferences and actions
